Telegram Bot

通过 BotFather 创建机器人
如何获取 Bot Token：
  1. 打开 Telegram，搜索 @BotFather
  2. 发送 /newbot，按提示给机器人起名
  3. BotFather 会返回一串 Token，格式如 123456789:ABCdef...
  4. 复制这串 Token 填入上方
→ 官方文档
如何获取你的 Telegram 用户 ID： 在 Telegram 中发消息给 @userinfobot，它会回复你的数字 ID。

微信接入方式

选择你要使用的接入类型
企业微信群机器人
个人微信（开放平台）
如何获取 Webhook URL：
  1. 在企业微信客户端中，打开目标群聊
  2. 点击右上角「…」→「添加群机器人」
  3. 创建新机器人，复制 Webhook 地址
个人微信接入（WeixinClawBot 官方插件）：
  1. 在运行 OpenClaw 的设备上，打开终端执行安装命令：
    npx -y @tencent-weixin/openclaw-weixin-cli@latest install
  2. 命令运行后会展示一个 QR 码
  3. 用你的微信扫描 QR 码完成授权
  4. 授权成功后 WeixinClawBot 插件即激活，无需填写 Token
⚠️ 此步骤需在终端手动完成，本向导会帮你启用对应插件配置。

飞书自建应用 Bot

支持私聊 + 群聊双向交互
⚠️ 配置前必须在飞书开放平台完成以下全部步骤，否则无法收到消息：
① 添加「机器人」能力  →  ② 开通权限  →  ③ 订阅事件（注意手动点开启）  →  ④ 发布版本
📋 完整配置流程（共 6 步）
  1. 访问 飞书开放平台 → 「创建企业自建应用」→「凭证与基础信息」复制 App ID 和 App Secret
  2. 左侧「添加应用能力」→ 找到 机器人 → 点「配置」启用
  3. 左侧「权限管理」→ 搜索并开通以下权限（缺一不可）：
    • im:message（发送消息，私聊+群聊必须）
    • im:message.p2p_msg:readonly（读取私聊消息）
  4. 左侧「事件与回调」→「事件配置」→ 点「添加事件」→ 搜索并添加 接收消息（im.message.receive_v1）
    ⚠️ 添加事件后，右侧「所需权限」里的每条权限需要单独点击「开启」按钮才生效！
    需开启：im:message.p2p_msg:readonly（私聊）+ im:message:group_at_msg:readonly（群聊 @ 机器人）
  5. 如需群聊：在飞书群聊中，点右上角「群设置」→「群机器人」→「添加机器人」→ 选择你的自建应用加入群
  6. 左侧「版本管理与发布」→「创建版本」→「提交发布」（企业内部应用需管理员审批）
→ 5分钟快速上手飞书机器人    → 机器人常见问题
